KONTROLLER

Cro-Mag Rally använder Input Sprocket för alla I/O-anslutningar. Som standard är spelet konfigurerat för att användas med tangentbordet, men du kan ändra kontrollerna för att använda vilket slags kontroll som helst som är kompatibelt med Input Sprocket. Det kan gälla joystick, gamepad, rattar osv.

ÄNDRA KONTROLLERNA

För att få upp dialogrutan för Input Sprocket, där du kan konfigurera kontrollerna, kan du när som helst trycka på F1. Eller kan du helt enkelt välja ALTERNATIV och sedan INSTÄLLNINGAR från spelets Menyruta.

 

VIKTIGT

Med de flesta tangentbord för Macintosh kan du bara trycka på högst 3 tangenter samtidigt. Det betyder att du måste ha någon annan kontroll, som ett andra tangentbord, gamepad, joystick osv., ansluten för att kunna spela med 2 spelare och delad skärm.

När du konfigurerar den andra kontrollen är det mycket viktigt att du gör det rätt. Nedan finns en skärmdump som visar hur man konfigurerar en Gravis Gamepad för Spelare 2. Du MÅSTE välja ”Spelare”: Två" och när du sedan ska ange kontrollerna, så se till att bara välja alternativen för "Spelare 2" i rullgardinsmenyn.

Det är lätt att av misstag välja kontrollinställningarna för “Spelare 1”, vilket medför att både tangentbordet och Gravis Gamepad samtidigt styr Spelare 1.Så var noggrann och se till att ange kontroller för rätt spelare och rätt kontrollapparat.
