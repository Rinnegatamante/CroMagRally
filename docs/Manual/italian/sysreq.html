REQUISITI DI SISTEMA

Per il corretto funzionamento di Cro-Mag Rally sono necessari i seguenti requisiti minimi di sistema:


CPU 233 Mhz
RAM 64MB
VRAM 6MB
3D ACCELERATION Qualsiasi scheda 3D in conformità con OpenGL- (potrebbe non funzionare correttamente con le vecchie schede ATI Rage II !)
OpenGL 1.2
Quicktime 4.1.2
MacOS 8.6
Game Sprockets 1.7.3
