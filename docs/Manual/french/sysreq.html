CONFIGURATION REQUISE

Cro-Mag Rally nécessite la configuration minimum suivante afin de fonctionner normalement :


PROCESSEUR 233 Mhz
RAM 64MB
VRAM 6MB
CARTE D'ACCÉLÉRATION 3D N'importe quelle carte 3D conforme à OpenGL (risque de ne pas fonctionner correctement avec une ancienne carte ATI Rage II!)
OpenGL 1.2
Quicktime 4.1.2
MacOS 8.6
Game Sprockets 1.7.3
