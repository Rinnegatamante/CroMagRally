MODE ENTRAÎNEMENT

Le mode entraînement vous permet de jouer sur n'importe quelle piste, à n'importe quelle période dans laquelle vous êtes en train de concourir, ou que vous avez achevée. Chaque fois que vous aurez complété une période en mode tournoi, les pistes de la période suivante s'ouvriront à vous de telle sorte que vous puissiez vous y entraîner.
