CHAT SAUVE-QUI-PEUT

Le Chat sauve-qui-peut est le contraire du Chat lépreux. Au lieu d'essayer d'éviter le poursuivant, dans ce jeu vous voulez être le joueur qui poursuit, vu que le premier joueur à être « le chat » pendant deux minutes sera le gagnant.


Le minuteur de chat


Le joueur qui poursuit est le seul à ne pas avoir de carte projetée, si bien qu'il ne peut voir où tout le monde se trouve.
