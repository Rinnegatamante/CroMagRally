LA QUÊTE DU FEU

Ce mode à plusieurs joueurs ressemble au traditionnel jeu de drapeau. Il y a deux équipes : les rouges et les verts. Il est préférable, pour jouer à ce jeu, de se trouver en nombre pair de sorte que les deux camps aient le même nombre de joueurs.

Chaque équipe commence la partie sur sa propre base. A côté de chaque base se trouvent les 6 torches de l'équipe adverse :


Un joueur de l'équipe rouge sur la base rouge


Le but est d'aller chercher vos torches dans la base du camp adverse et de les ramener à votre base. La base de l'équipe rouge a un drapeau rouge et les torches de cette équipe brûlent avec une lueur rouge. La base de l'équipe verte a un drapeau vert et les torches de cette équipe brûlent avec une lueur verte. Les rouges doivent donc trouver toutes les torches rouges et les ramener à la base au drapeau rouge. Vice versa pour les verts.


Un équipier rouge transportant une torche rouge


Vous pouvez ramasser les torches de votre adversaire et les cacher n'importe où sur le terrain. Pour ramasser une torche, il suffit de passer dessus. Pour déposer une torche, appuyer sur une touche de jet. Pour forcer un autre joueur à laisser tomber sa torche, vous pouvez soit lui rentrer dedans, soit le frapper avec une des armes explosives. Pour déposer une torche à votre base il suffit de toucher la base et la torche y sera déposée automatiquement.

La première équipe à ramener toutes ses torches à sa base est la gagnante.
