NETZWERK-SPIEL

Cro-Mag Rally kann über ein lokales Netzwerk (LAN) mit 2 bis 6 Spielern gespielt werden. Das Spiel benutzt die NetSprockets, um ein Netzwerk-Spiel zu konfigurieren, so daß Sie mit der Bedienung vielleicht schon aus anderen Spielen vertraut sind.

Um ein Netzwerk-Spiel starten zu können, muß eine Person der Anbieter (Host) sein und alle anderen Spieler sind die Teilnehmer (Clients), die dem Spiel beitreten, das vom Anbieter erzeugt wird. Der Anbieter wählt den Spiel-Modus aus und die zu fahrende Strecke. Aus Geschwindigkeitsgründen sollte der Spieler der Anbieter sein, der den schnellsten Rechner hat. Teilnehmer können das Netzwerk-Spiel verlassen und das Spiel läuft ohne sie weiter. Wenn der Anbieter jedoch das Netzwerk-Spiel verläßt, ist das Spiel beendet.


ANBIETEN

Wenn Sie ein Netzwerk-Spiel anbieten, bekommen Sie folgenden NetSprocket-Dialog zu sehen:

Net Sprocket erlaubt das Spielen über eine AppleTalk- oder TCP/IP-Verbindung. Wir empfehlen, nicht über TCP/IP zu spielen, da das Spiel dann sehr langsam wird. Wenn Sie es aber versuchen möchten, besteht die Möglichkeit dazu. Die Einstellung "Play on this machine" muß gesetzt sein.

Wenn Sie auf OK klicken, wird das Spiel im Netzwerk eingerichtet und andere Spieler können dem Spiel nun beitreten....


TEILNEHMEN

Bis zu 5 Teilnehmer können einem vom Anbieter erzeugten Netzwerk-Spiel beitreten. Wenn Sie dem Spiel beitreten, bekommen Sie folgenden NetSprocket Dialog zu Gesicht:

Im Bereich "Choose a Game" sehen Sie eine Liste aller angebotenen Spiele im Netzwerk, die Teilnehmer akzeptieren. Wählen Sie einfach das Spiel aus, an dem Sie teilnehmen möchten, geben Ihren Namen ein und klicken Sie auf OK.

Wenn Sie den Anbieter im Fenster nicht sehen, überprüfen Sie bitte, ob alle Spieler die gleiche Version von CroMag-Rally verwenden. Neuere Versionen des Spiels sind vielleicht nicht mit älteren kompatibel.

Nachdem alle Spieler dem Spiel beigetreten sind, startet der Anbieter das Spiel und Sie können sich bereit machen zu spielen!
