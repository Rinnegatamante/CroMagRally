DAS SPIEL

Cro-Mag Rally besitzt viele verschiedene Spiel-Modi und kann von 1 bis 6 Spielern gespielt werden. Das Spiel findet in der Steinzeit, Bronzezeit und Eisenzeit statt. Sie müssen ein Zeitalter vollendet haben, bevor Sie Strecken in späteren Zeitaltern fahren dürfen. Wenn Sie das Spiel frisch installiert haben, dürfen Sie nur die Steinzeit spielen. Wenn Sie das Turnier in der Steinzeit gewonnen haben, eröffnet sich für Sie die Bronzezeit und zusätzlich bekommen Sie neue Autos zu Auswahl. Wenn Sie die Bronzezeit vollendet haben, wird Ihnen die Eisenzeit angeboten.
