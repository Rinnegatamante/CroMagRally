TRAININGS-MODUS

Im Trainings-Modus können Sie jede Strecke in jedem Zeitalter fahren, in dem Sie sich gerade befinden oder das Sie vollendet haben. Jedesmal, wenn Sie ein Zeitalter im Turnier-Modus vollendet haben, bieten sich die Strecken des nächsten Zeitalters an, so daß Sie auf diesen trainieren können.
