SYSTEM REQUIREMENTS

Cro-Mag Rally needs the following minimum system requirements to function properly:


CPU 233 Mhz
RAM 64MB
VRAM 6MB
3D ACCELERATION Any OpenGL-compliant 3D Accelerator (will not work with old ATI Rage II accelerators!)
OpenGL 1.1.3
Quicktime 4.1.2
MacOS 8.6 - OS X*
Game Sprockets 1.7.3


NOTE to Rev A iMac and "beige" G3 owners:

Both the original Rev A iMacs and the old beige G3's came installed with ATI Rage II 3D accelerators. Those accelerators are not OpenGL compliant and are too slow for Cro-Mag Rally to be played on.

Biege G3 owners simply need to install a more recent 3D acclerator, we stronly recommend the ATI Rage 128 or better. The Blue and White G3's have Rage 128's which will run the game excellently as is.

Rev A iMac owners will not be able to play the game unless they have a Voodoo2 card installed from Micro Conversions. This card is no longer made, so if you don't already have one installed then it is unlikely that you will be able to find one. All other iMac's (Rev B and up) will play the game just fine since they have Rage Pro or Rage 128 accelerators.


*OS X Compatibility

This version of Cro-Mag Rally has been Carbonized which menas that it will technically run on OS X. However, since this version shipped before OS X shipped, not all components of OS X were complete at publication time and thus some fundamental features do not work: Full-Screen support, I/O, and Networking.

We plan on making Cro-Mag Rally fully OS X compatible and will release free updates as soon as that happens. Please check the Pangea Software web site for more information and updates.
